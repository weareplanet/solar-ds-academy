# Exercise 5.1: Product Pitch Prototype (Capstone)

**Goal:** Build a complete prototype for a realistic product idea in under 1 hour.

## Instructions

1. **Choose a product idea** (pick one):
   - Merchant onboarding portal
   - Risk & compliance dashboard
   - Settlement tracker
   - …or propose your own!

2. **Plan 5–7 screens** — write down what each page should contain

3. **Build all screens** using @Solar
   - Start with `@solar /initiate [Your Idea]`
   - Add each page by describing it

4. **Refine** with visual context until presentation-quality

5. **Validate** with `@solar /validate`

6. **Publish** via `@solar /share`

## Deliverable

Upload via [Microsoft Forms](https://forms.cloud.microsoft/e/AsPkgKDKYu):
- The **public URL** of your prototype
- A **1-paragraph write-up**: What you built, how many pages, how long it took

## Success Criteria

- A polished, multi-page prototype (minimum 5 pages)
- All pages reachable from sidebar navigation
- Could be shown to a stakeholder in a meeting
- `/validate` passes with no critical issues
