# Exercise 3.1: Design Review Simulation

**Goal:** Refine a page through 5 iterative prompts using visual context.

## Instructions

1. Create a dashboard page (reuse from Exercise 2.1 or create a new one)
2. Type `#browser` in chat to attach a screenshot
3. Ask: *"the stat cards look too small — make them wider and add icons"*
4. Use **Select Element** on the chart → *"change this to a line chart"*
5. Continue refining — **minimum 5 iterations**

## Deliverable

Upload via [Microsoft Forms](TODO_FORMS_LINK):
- **Before** screenshot (after initial creation)
- **After** screenshot (after 5+ refinements)
- List of the **5 prompts** you used

## Success Criteria

- Visible improvement between before and after
- At least 5 distinct refinement prompts were used
- At least one prompt used `#browser` or Select Element
