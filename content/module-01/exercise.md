# Exercise 1.1: Hello Solar

**Goal:** Create your first app with one command.

## Instructions

1. Open **VS Code**
2. Open the Copilot Chat panel (`Cmd+Shift+I`)
3. Type: `@solar /initiate My First App`
4. Wait for the project to scaffold, install dependencies, and open a live preview
5. **Screenshot** the running app in the built-in browser

## Deliverable

Upload your screenshot via [Microsoft Forms](TODO_FORMS_LINK).

## Success Criteria

- A running app is visible in VS Code's built-in browser
- The app has a sidebar navigation
- A "Get Started" page is visible

## Tips

- If the preview doesn't open automatically, check the terminal for errors
- Make sure Node.js is installed (`node --version` in terminal)
- If Copilot Chat doesn't recognize `@solar`, reinstall the extension
